# HRMS AI Agent Requirements

## Project Objective

Develop an AI-powered HR assistant capable of answering employee questions and performing HR-related actions through natural language conversations.

## Functional Requirements

### Leave Management

* View leave balance
* Apply for leave
* Cancel leave request
* View leave history

### Attendance Management

* View attendance records
* Calculate attendance percentage
* Check late arrivals

### Payroll Support

* View salary details
* Download payslips
* View deductions
* View tax information

### Employee Information

* View employee profile
* Update contact information
* View department details

### Recruitment Support

* View candidate information
* Schedule interviews
* Notify interviewers

## Non-Functional Requirements

* Secure authentication
* Fast response times
* Scalable architecture
* Audit logging
* Data privacy compliance

## Technology Stack

Frontend:

* React or Streamlit

Backend:

* Python

AI Layer:

* OpenAI API

Database:

* PostgreSQL or JSON prototype

Version Control:

* Git and GitHub
